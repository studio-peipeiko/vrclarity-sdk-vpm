# VRClarity SDK 利用規約 (Terms of Service)

**最終更新日**: 2026年7月16日
**バージョン**: 1.4.1

この利用規約（以下「本規約」）は、studio peipeiko（以下「当方」）が提供する VRClarity SDK（以下「本 SDK」）の利用条件を定めるものです。本 SDK を使用することで、本規約に同意したものとみなされます。

## 1. サービス概要

VRClarity SDK は、VRChat ワールドの利用状況に関する匿名の集計値を VRClarity へ送信する、ワールドに組み込む UdonSharp 製コンポーネントです。ワールド制作者は本 SDK を使用することで、以下の匿名の集計データを自動送信し、VRClarity のダッシュボードで確認・分析できます：

- 滞在時間・移動距離のマイルストーン
- 訪問回数のバケット集計
- プラットフォーム区分（PCVR / Desktop / Quest / Android / iOS）
- プレイヤー数の時系列集計

### 1.1 訪問プレイヤーへの告知（重要）

ワールド制作者は、本 SDK を導入したワールドにおいて、訪問プレイヤーに対し「VRClarity を利用して個人を特定しない匿名の統計データを送信していること」を告知することを**強く推奨**されます。本 SDK は、Tracker を作成すると訪問プレイヤーへデータ送信を案内する「Notice Panel」を既定で自動生成・配置します。ワールド制作者は、この Notice Panel を訪問プレイヤーの見える場所に表示・維持する等により、上記の告知を確保してください。データの取り扱いの詳細は、[プライバシーポリシー](https://vrclarity.net/docs/privacy)および[サービス利用規約](https://vrclarity.net/docs/terms)をご参照ください。

## 2. データの所有権

### 2.1 送信データの所有権

**VRClarity に送信されたメトリクスデータを集計・統計化したデータセット、およびそこから派生する成果物に関する権利は、VRClarity（当方）に帰属します。** ワールド制作者は、本 SDK を使用することでデータの送信・集計利用に同意しますが、集計・統計化されたデータセットについて権利を主張しません。個々のデータ主体（訪問プレイヤー等）がデータ保護法令に基づき有する権利は、これにより一切制限されません（次段参照）。

なお、本条にいう「所有権」は、集計・統計化されたデータセットおよびそこから派生する成果物に関する権利を指すものであり、プライバシーポリシーおよび適用されるデータ保護法令（EU一般データ保護規則（GDPR）、日本の個人情報保護法等）に基づき各データ主体（訪問プレイヤー等）が有する権利を制限・否定するものではありません。当該データに個人情報・仮名化情報が含まれる範囲における取り扱いは、[プライバシーポリシー](https://vrclarity.net/docs/privacy)に従います。

### 2.2 データの利用権

- **ワールド制作者**: 自身のワールドに関する統計情報を VRClarity のダッシュボードで閲覧できます
- **VRClarity**: 送信されたデータを以下の目的で利用できます：
  - サービスの提供・改善
  - 匿名化・集計された統計情報の公開
  - 研究・分析目的での利用
  - サービスの紹介・統計情報の公表

### 2.3 データの第三者提供

VRClarity は、匿名化・集計されたデータを統計情報として第三者に提供する場合があります。ただし、個人を特定できる情報や特定のワールドを識別できる情報は含まれません。

## 3. 禁止事項

本 SDK の利用にあたり、以下の行為を**厳重に禁止**します。違反が確認された場合、SDK の利用停止、データの削除、法的措置を取る場合があります。

### 3.1 エンドポイントの変更・改変

- 本 SDK が送信するデータのエンドポイント URL を変更すること
- VRClarity のサーバー以外にデータを送信するように改変すること
- ネットワーク設定やプロキシを使用してデータの送信先を変更すること

### 3.2 データの横流し・不正利用

- 送信されたデータを第三者に提供・販売・譲渡すること
- 本 SDK を改変して独自のデータ送信システムとして利用すること
- VRClarity のインフラを不正に利用すること
- スクレイピングやクローリングによるデータの大量取得

### 3.3 SDK の悪用

- 取得した暗号化キー・API キーを第三者に公開・譲渡すること
- セキュリティ機能を無効化・回避して VRClarity のサービスを不正利用すること

### 3.4 不正アクセス・サービス妨害

- VRClarity のサーバーに不正アクセスを試みること
- DDoS 攻撃やその他のサービス妨害行為
- 虚偽のデータや不正なデータを意図的に送信すること
- 過度なリクエストによるサーバー負荷の発生

### 3.5 商標・知的財産権の侵害

- VRClarity の商標・ロゴを無断で使用すること
- 本 SDK を再配布・再販売すること（VPM レジストリ経由の正規配布を除く）
- 本 SDK の著作権表示を削除・改変すること
